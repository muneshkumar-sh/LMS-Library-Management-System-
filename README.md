# Library Management System

A Java-based Library Management System developed as a Data Structures and Algorithms (DSA) semester project.

## Features
- Login-based system with Student and Librarian roles
- Role-based access control
- Book management (Add, Edit, Delete, Search, Sort)
- Issue and return books
- Undo functionality using Stack (LIFO)
- MySQL database integration
- Java Swing based GUI

## Technologies Used
- Java
- Java Swing
- MySQL
- Data Structures and Algorithms (DSA)

## How to Run
1. Import the project into NetBeans / IntelliJ / Eclipse
2. Configure MySQL database using provided SQL file
3. Run `Main.java`
